# Zero trust microservice

A production-grade microservices environment deployed on a bare-metal Kubernetes (K3s) cluster, designed to demonstrate enterprise-level Zero-Trust security principles. This project features a decoupled backend (Go) and frontend (Node.js) communicating exclusively through an Istio Service Mesh with strict mutual TLS (mTLS) enforced. To completely eliminate hardcoded infrastructure secrets, HashiCorp Vault is deployed via Helm to dynamically inject database credentials directly into the application pods at runtime. Authentication is handled natively through ephemeral Kubernetes ServiceAccounts, adhering to the principle of least privilege. The entire development workflow, including dependency management and tooling, is entirely deterministic and reproducible using Nix Flakes.
